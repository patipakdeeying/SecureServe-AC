Installation Guide

Follow these steps to get SecureServe up and running on your server:

  1. Set up logs webhooks for event logging.
  2. Initial Ban Notice: You may see a ban reason for "Trigger Event with an executor (name of the event)" upon first install.
  3. Open the configuration file and copy the event name causing the ban.
  4. Add this event name to the whitelisted events list in the config.
  5. This resolves the issue caused by encrypted and decrypted scripts running the same events.
  6. Adjust explosion max/min settings to fit your server’s needs. Don’t simply whitelist all staff events.

Pro Tip: Keep your config up to date, especially if you use other scripts that might conflict with event triggers.
